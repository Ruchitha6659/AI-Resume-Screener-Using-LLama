# AI Resume Screener using LLaMA

## Overview
AI Resume Screener is an intelligent system that analyzes and ranks candidate resumes based on job descriptions using Natural Language Processing and LLaMA-based models. The system helps automate the resume screening process by identifying relevant skills and qualifications.

## Features
- Automated resume analysis
- Skill and keyword extraction
- Resume ranking based on job description
- NLP-based text processing
- AI-assisted candidate shortlisting

## Technologies Used
- Python
- Generative AI (LLaMA)
- Natural Language Processing (NLP)
- Machine Learning

## Project Workflow
1. Upload resume
2. Extract text from resume
3. Process text using NLP techniques
4. Compare resume content with job description
5. Rank resumes based on relevance

## Future Improvements
- Improve resume ranking accuracy
- Add support for multiple job roles
- Integrate a web interface
